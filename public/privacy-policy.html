FlexReader Privacy Policy

Last updated: February 11, 2026

This Privacy Policy explains how FlexReader handles data when you read books, import files, browse Project Gutenberg, and use optional online AI features.

1. Data We Store On Your Device

  • Imported books and generated metadata (title, language, sections, chapters).
  • Dictionary data (offline dictionaries and optional generated entries).
  • Reading progress, shelf assignments, reader settings, and local reading stats.
  • Cached Gutenberg bookshelf and book-list snapshots for faster browsing.
  • Terms consent state and app-local preference data.

Storage is local to your device (IndexedDB and device preferences).

2. Network Requests

  • Project Gutenberg: bookshelf browsing, catalog search, metadata lookups, and EPUB downloads.
  • OpenAI / DeepSeek (optional): if enabled, the app sends selected text/context for: translation, dictionary generation, quiz-question generation, and optional on-demand LLM word meanings.

3. API Keys

  • You can enter API keys in the pipeline UI for optional AI features.
  • Keys are used only to call the provider you selected.
  • Keys are not stored as account/profile data and are not uploaded to a FlexReader backend.

4. Data Sharing

FlexReader does not sell personal data. FlexReader has no first-party analytics SDK and no ad SDK in this build. Data leaves the device only for features you explicitly use:

  • To Gutenberg for catalog and EPUB retrieval.
  • To your selected AI provider for optional AI processing (translation / dictionary / questions / LLM meaning).

5. AI-Generated Content and Reporting

AI-generated output may be inaccurate or unsuitable. The app provides in-app report/flag actions on AI-generated surfaces so users can report problematic output.

6. Children

FlexReader is a general reading app. Parents/guardians should supervise when children use network-based features. Do not enter sensitive personal information into imported files or prompts.

7. Retention and Deletion

  • You can delete imported books and dictionaries from inside the app.
  • You can clear app storage by uninstalling the app or clearing app data in Android settings.

8. Third-Party Services

  • OpenAI: openai.com privacy policy
  • DeepSeek: platform.deepseek.com
